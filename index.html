Селекторы бывают:

  • по названию элемента
  • по id
  • по классу
  • по нескольким классам
  • по вложенности
  1. h1-будут выбраны все элементы с тэгом "h1"
  2. #test-будут выбраны все элементы c id "test"
  3. .test будут выбраны все элементы с атрибутом class="test"
  4. .test.test-2 будут выбраны все элементы с атрибутом class="test.test-2
  5. a>b будут выбран элемент "b" лежащий в "a"
  6. * будет применятся ко всем элементам на странице
